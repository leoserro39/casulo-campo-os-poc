# Mapa de Dados Computaveis

Descreve fontes, sinais computaveis, dominios alimentados, lacunas e uso permitido/bloqueado.

## Campos minimos

source_id, nome, tipo, dominio dono, dominios suportados, formato, frequencia, atualizacao, confiabilidade, sensibilidade, riscos, sinais computaveis, sinais ausentes, uso permitido, uso bloqueado.

## Estados de uso

USE, USE_WITH_CAUTION, USE_AS_INDICATION_ONLY, NEED_CONFIRMATION, DISCARD, BLOCK_RECOMMENDATION.
