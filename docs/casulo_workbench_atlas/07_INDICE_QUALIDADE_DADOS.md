# Indice de Qualidade de Dados

Mede se uma fonte pode balizar diagnostico, delta, gate ou solucao.

## Dimensoes

completude, atualizacao, consistencia, rastreabilidade, confiabilidade da fonte, cobertura de dominio, independencia da evidencia, contradicoes, sensibilidade/risco, estrutura computavel.

## Faixas

- 0.00 a 0.30: baixa qualidade.
- 0.31 a 0.60: uso controlado.
- 0.61 a 0.80: qualidade aceitavel.
- 0.81 a 1.00: alta qualidade.

Nenhum dado entra em recomendacao forte sem qualidade minima.
