# PROD-1781..1820 - Graph-Based Opportunity Ranking

This phase converts the business solution ensemble into a graph-based opportunity ranking.

It does not perform final calibration.
It does not connect GPT.
It does not call GPT.
It does not call Codex.
It does not connect to production.
It does not authorize client-facing claims.

## Purpose

Create a graph-style ranking that connects:

- company profiles;
- service packages;
- solution types;
- recommendation types;
- risks;
- gates;
- monitoring needs;
- governance needs;
- opportunity scores.

## Why graph ranking matters

The ensemble scores individual cases.

The graph ranking identifies which business patterns concentrate the most CASULO value across cases.

This helps prioritize:

- first pilots;
- strongest service packages;
- safest sales narratives;
- recurring monitoring opportunities;
- future calibration focus.

## Boundary

This is a graph-style analytical projection from synthetic/provisional taxonomy data.

It is not a production graph import.
It is not a final market claim.
It is not threshold calibration.
