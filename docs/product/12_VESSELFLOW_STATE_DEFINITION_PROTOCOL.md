# VesselFlow State Definition Protocol

Use VesselFlow as the complex vertical to test operational state definition across vessels, PVQ/Q88, documents, contracts, cargo, platform, qualification, synchronization and nomination.

## Required output layers

1. State Snapshot.
2. Operational Graph.
3. Domain Map.
4. Contract Map.
5. Nomination Flow.
6. Gate Matrix.
7. Evidence Manifest.
8. Risk/Fragility Index.
9. Delta Recommendations.
10. Cube/Cupula State.
11. Cockpit Replay.
12. Report.

Default rule: the system must not execute a nomination automatically.
