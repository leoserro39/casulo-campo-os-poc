# VesselFlow Import Protocol

## Purpose

Prepare VesselFlow for controlled complex operational state definition.

## Required input from user

- dataset/workbook;
- expected nomination flow;
- qualification rules;
- known blocking rules.

## Expected product output

- State Snapshot;
- Operational Graph;
- Domain Map;
- Contract Map;
- Nomination Flow;
- Gate Matrix;
- Evidence Manifest;
- Risk/Fragility Index;
- Delta Recommendations;
- Cube/Cupula State;
- Cockpit Replay;
- Report.

## Default policy

The system does not execute nomination automatically.

It defines state, identifies gates and prepares a human review path.
