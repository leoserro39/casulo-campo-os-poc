# CASULO Workbench — MVP Scope

## MVP goal

Create a navigable product runtime for the controlled internal POC outputs.

## In scope

- Case list.
- Case workspace.
- Intake summary.
- Evidence summary.
- Diagnostic summary.
- Cube/Cupula state view.
- Cockpit story replay.
- Micrograph event timeline.
- Report viewer.
- Human review console.
- Gate status.
- Exportable internal report.

## Out of scope for this phase

- Multi-tenant SaaS.
- Production authentication.
- Billing.
- Full 3D polished Cube.
- Real client production use.
- Automated implementation.
- External client-facing claims.

## MVP success criteria

The operator must be able to open the product locally and explain:

1. What entered.
2. What evidence exists.
3. What state was computed.
4. What risk was measured.
5. What delta was recommended.
6. What gate blocks external use.
7. What the Cube/Cupula represents.
8. What the next safe action is.
