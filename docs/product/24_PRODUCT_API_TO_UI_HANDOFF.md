# Product API to UI Handoff

The next UI should consume these endpoints:

- `/api/product/status`
- `/api/verticals`
- `/api/verticals/{vertical_id}`
- `/api/verticals/{vertical_id}/state-request`
- `/api/vesselflow/import-manifest`
- `/api/reports`

Recommended first UI:

- Home
- Vertical list
- VesselFlow case workspace
- Cube/state request panel
- Import manifest panel
- Report panel
