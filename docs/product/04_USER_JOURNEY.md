# CASULO Workbench — User Journey

## Persona 1 — Operator / Founder

The operator opens a case and reviews the operational state.

Flow:

1. Open Home.
2. Select controlled case.
3. See POC status.
4. Open Cube/Cupula state.
5. Review Evidence.
6. Review Diagnostic.
7. Replay story.
8. Review micrograph timeline.
9. Check gates.
10. Export report or request more evidence.

## Persona 2 — Internal Reviewer

The reviewer decides whether the case can move closer to client-facing review.

Flow:

1. Open internal handoff.
2. Check audit status.
3. Check blocked actions.
4. Confirm whether more evidence is needed.
5. Keep or modify human review decision.

## Persona 3 — Future Client Viewer

Not enabled yet.

The current product must block client-facing review until explicit approval.
