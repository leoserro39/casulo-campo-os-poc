# Anonymization Checklist

Before running controlled intake, confirm:

- [ ] Client/company identifiers removed or replaced.
- [ ] Names removed or replaced.
- [ ] Personal contact data removed.
- [ ] Financial sensitive details generalized when not required.
- [ ] Dates generalized when exact date is not required.
- [ ] Free text reviewed for accidental personal data.
- [ ] Evidence classification set.
- [ ] Allowed use set.
- [ ] Human review required for confidential/sensitive sources.
