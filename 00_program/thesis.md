# Tese do Programa

O CASULO Campo OS testa se uma IA assistida por metodo consegue transformar documentos soltos, observacoes de campo e dados simples de pequenos negocios em uma malha operacional navegavel.

A tese pratica e: uma melhoria pequena e validada por vez gera mais valor do que uma resposta grande sem evidencia.

A unidade de trabalho e:

Entrada -> dominio -> evidencia -> problema -> micro-solucao -> execucao -> metrica -> delta.
