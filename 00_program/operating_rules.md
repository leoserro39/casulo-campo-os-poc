# Regras de Operacao

- Uma rodada resolve uma dor por vez.
- Nenhuma promessa de resultado sem evidencia.
- Dados sensiveis nao devem ir para case publico.
- Toda solucao precisa ter responsavel, metrica e prazo.
- Toda saida deve apontar fontes ou lacunas.
- O chat e interface, nao memoria canonica.
- O commit/delta validado e a unidade de persistencia.
