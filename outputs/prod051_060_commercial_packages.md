# Commercial Packages and AI Program

## Summary
Suggested service packages for SMEs and technical teams.

## Suggested Offers
- **CASULO Operational Diagnosis**: Map current state, evidence gaps, risks and first gates.
- **Cubo PME Starter**: Install a first operational cube for one recurring process.
- **TIC/SI State Mesh**: Map applications, integrations, incidents, access, changes and gates.
- **Software Review Gate**: Review existing software and produce a controlled improvement plan.
- **Solution Factory Sprint**: Transform approved deltas into governed software tasks, tests and delivery evidence.
- **AI Program Operating Layer**: Structure AI adoption around state, evidence, governance, value deltas and safe execution.
