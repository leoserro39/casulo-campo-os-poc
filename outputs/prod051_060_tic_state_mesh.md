# TIC/SI State Mesh

## Summary
The TIC/SI mesh connects applications, services, integrations, access, incidents, changes, data, suppliers and deployments into a computable operational state.

## Gates
- `security_gate`
- `production_gate`
- `support_gate`
- `continuity_gate`
- `data_gate`
- `integration_gate`
- `change_gate`
- `rollback_gate`

## Blocked Actions
- `automatic_nomination`
- `client_facing_claim`
- `implementation_execution`
- `production_activation`
