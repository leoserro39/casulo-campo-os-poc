# Software Review Gate

## Summary
The software review gate evaluates whether an existing or planned solution is safe, maintainable, testable and operationally ready.

## Review Dimensions
- `architecture`
- `security`
- `documentation`
- `tests`
- `deployment`
- `observability`
- `data model`
- `integration model`
- `operational ownership`
- `rollback`

## Blocked Actions
- `automatic_nomination`
- `client_facing_claim`
- `implementation_execution`
- `production_activation`
