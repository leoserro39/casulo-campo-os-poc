# Real Case 001 - Intake Checklist v0.3

Before any execution gate, confirm:

- [ ] Domain declared.
- [ ] Task objective declared.
- [ ] Input sanitized.
- [ ] No PII.
- [ ] No secrets or credentials.
- [ ] No production command or real-world side effect.
- [ ] Evidence packet filled.
- [ ] Source boundaries filled.
- [ ] Allowed actions reviewed.
- [ ] Blocked actions reviewed.
- [ ] Expected gate declared.
- [ ] Human review rubric accepted.
- [ ] Claim boundary acknowledged.
- [ ] Stop criteria accepted.

If any item is unchecked, keep execution blocked.
